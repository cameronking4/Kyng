Devil's Cup

Mobile Game / App

While I was studying abroad in London (Fall 2018), my friends from Duke and I would play King's Cup and eventually we created our own version of the game with Duke related prompts. I decided to turn this game into an app for all Duke students to play and submit to. I also curate the instagram handle @drinking.devils reposting the best prompts that users submit.

Tools Used
  • React Native CLI
  • Expo Client
  • HTML / CSS / Javascript
  • Adobe Photoshop

    • The code for this project is open source
