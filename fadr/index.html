What is Fadr

Fadr is a mobile solution for finding a reliable, trustworthy barber wherever you may be.

The one and only solution for all your grooming needs, from booking to execution.

For the first time ever, you can go anywhere and know that you ensured a quality haircut. Given our rigorous barber approval process and the ability to view barber profiles before reserving your seat, you can rest assured knowing that you're in the right hands.

With the abilies to give direct instructions to your barber of choice, explore trending hairstyles, view photo reviews and instantly book an appointment, Fadr is an app made for a winner.

Core Features

Barbershop Review Compiler

Suppose you were looking for a new barber. Using traditional methods, you would search for a shop near you, using Google and Yelp.

We have a developed a web application to do one search and find shops within a specified radius of a user using the Google Places and Yelp API; it returns to you a list of results from both Yelp and Google in which you can browse through ratings, photos, locations and reviews of barberhsops near you. However, Fadr hopes to make this process much easier.


View Project


This project is open source. Simply download the zip file and open the "index.html" file in your browser to run the application
